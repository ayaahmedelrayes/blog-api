# Blog API

Express.js + MongoDB + JWT + ImageKit

## Setup
1. Clone the repo
2. Run: npm install
3. Copy .env.example to .env and fill in values
4. Run: npm run dev

## API Endpoints
- POST /api/auth/register
- POST /api/auth/login
- GET/POST/PUT/DELETE /api/posts
- GET/PUT/DELETE /api/users
- POST /api/groups
